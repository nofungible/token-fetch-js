  • Object
Properties:

 Date constraint limiting tokens to after the provided timestamp.

 Date constraint limiting tokens to before the provided timestamp.

 Token ID constraint for token fetch. IDs are contract specific. Multi-contract providers are expected to manage contract-id relationships.

 Wallet address constraint for token creator.

 Number of results to take. Default is 50.

 Wallet address constraint for token owner.

 Number of results to skip. Defualt is 0.

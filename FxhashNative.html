Class: FxhashNative

FxhashNative(key)

Native fx(hash) indexer. Lacks many expected query operators since it was built specifically for fx(hash). Simple things like gathering feeds, or user collections, is fine. For more complex querying use TezTok, or ObjktcomNative.

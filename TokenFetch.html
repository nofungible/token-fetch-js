Class: TokenFetch

TokenFetch(providerSet)

Provide a set of token providers and fetch normalized token data using a simple query interface.

Constructor

new TokenFetch(providerSet)

TokenFetch requires a set of providers to fetch tokens from. Generally speaking, this will be a set of indexers. You can access built-in providers to utilize via module:TokenFetchJS.providers.
